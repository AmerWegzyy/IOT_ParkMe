# Assets

Project media that isn't source code.

## Poster (required for final submission)

Add a photo (or PDF export) of the final project poster here, e.g.
`assets/poster.jpg`, then embed it in the root `README.md` under the **Poster**
section:

```markdown
![ParkMe poster](assets/poster.jpg)
```

## Other assets (optional)

Sound files, project photos, 3D enclosure files (e.g. a printed box for the
boards), etc. can also go here.

> The wiring/connection diagram lives in
> [`../Documentation/images/`](../Documentation/images/) alongside the hardware
> documentation.
